# Contributing

感谢你参与改进 WPS 灵犀皮肤管理器。

## 提交问题

请优先使用仓库的 Issue 模板，并尽量提供：

- macOS 与 WPS 灵犀版本；
- 日间或夜间模式；
- 出现问题的皮肤；
- 完整窗口截图；
- 是否可以稳定复现。

提交截图前，请隐藏历史会话、项目名和其他个人信息。

## 提交代码

1. 从 `main` 创建功能分支。
2. 保持修改范围单一，不提交主题配置、日志、临时抠图文件或本机应用包。
3. 运行 `npm run check`。
4. 在日间和夜间模式中测试至少一套普通主题和一套带挂件主题。
5. 提交 Pull Request，并说明变更原因、视觉影响和验证方式。

## 主题素材

只提交自己拥有权利或明确获得授权的素材。涉及品牌、IP、角色和商标时，请同时说明来源和使用边界。
